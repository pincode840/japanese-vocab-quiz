# 일본어 단어 맞추기

JLPT N5·N4·N3·N2 어휘를 한자, 읽기, 문장 빈칸 문제로 학습하는 웹 퀴즈입니다.

## 바로 실행

[일본어 단어 맞추기 앱 열기](https://pincode840.github.io/japanese-vocab-quiz/)

## 주요 기능

- JLPT `N5·N4`, `N3`, `N2` 난이도 선택
- `한자 → 읽기`, `한자 → 히라가나 조합`, `읽기 → 한자`, `문장 빈칸 → 한자` 모드
- 문제마다 선택지 `4개`, `6개`, `8개` 설정
- 일반 및 문장 문제 수를 `10개부터 해당 데이터 전체`까지 선택
- 키보드 숫자키와 숫자패드 `1~8`로 답 선택
- 정답을 선택하면 단어의 읽기와 뜻 표시
- 오답이면 선택한 답의 단어·읽기·뜻과 올바른 정답 표시
- 틀린 문제는 몇 문제 뒤 자동으로 다시 출제
- 오답 이력이 있는 문제를 다음 회차에서 우선 출제
- 난이도와 학습 모드별 회차·최근 정답률·누적 정답률 분리 기록
- ON/OFF 가능한 100문제 시험 모드와 문제별 제한시간
- 상단 `読` 아이콘을 누르면 언제든지 시작 화면으로 이동

## 문제 수

| 난이도 | 일반 단어 문제 | 문장 빈칸 문제 |
|---|---:|---:|
| N5·N4 | 480개 | 480개 |
| N3 | 한자 어휘 1,323개 | 500개 |
| N2 | 100개 | 100개 |

## 오답 학습 방식

### 일반 단어 모드

틀린 선택지와 함께 해당 선택지의 단어·읽기·뜻을 보여줍니다. 올바른 정답과 뜻도 확인할 수 있으며, 틀린 문제는 잠시 뒤 다시 나옵니다.

예시: `선택한 답 帽子（ぼうし） · 모자`

### 문장 빈칸 모드

1. 첫 번째 오답: 선택한 답의 읽기와 뜻을 표시하고 문장에 후리가나를 붙여 같은 문제에 다시 답하게 합니다.
2. 두 번째 오답: 선택한 답, 정답, 단어 뜻과 문장 해석을 모두 보여줍니다.
3. 두 번 틀린 문장은 몇 문제 뒤 다시 출제됩니다.

## 한자 → 히라가나 조합 모드

- 문제의 한자를 보고 히라가나 타일 9개를 순서대로 눌러 읽기를 완성합니다.
- 정답에 필요한 모든 히라가나와 오답용 글자를 섞어서 제시합니다.
- 같은 히라가나 타일을 여러 번 사용할 수 있어 중복 글자가 있는 읽기도 입력할 수 있습니다.
- 화면 버튼 외에 숫자키 `1~9`로 글자 선택, `Backspace`로 한 글자 지우기, `Delete` 또는 `Esc`로 전체 지우기, `Enter`로 제출할 수 있습니다.
- 일반 학습의 오답 재출제와 시험 모드의 10초 제한시간·누적 오답 표시·3회차 출제 제외 기능이 함께 적용됩니다.

## 시험 모드

- 시작 화면에서 시험 모드를 `ON/OFF`로 전환할 수 있습니다.
- 시험은 문제 수가 100개로 고정됩니다.
- 객관식 단어 문제는 문제당 7초, 한자→히라가나 조합 문제는 10초, 문장 문제는 13초의 제한시간이 적용됩니다.
- 제한시간 안에 답하지 못하면 시간 초과 오답으로 처리하고 정답을 보여줍니다.
- 시험에서 틀린 문제에는 해당 시험 문제의 누적 오답 횟수를 표시합니다.
- 시험에서 맞힌 문제는 같은 `난이도 + 학습 모드`의 다음 3개 완료 회차 동안 시험 출제 대상에서 제외됩니다.
- 보호 중인 문제 때문에 출제 가능한 문제가 100개보다 적으면 시험을 시작하지 않고 현재 출제 가능 수를 안내합니다. 일반 학습 회차를 진행하면 보호 기간을 넘길 수 있습니다.

## 학습 기록

- `난이도 + 학습 모드` 조합별로 완료 회차, 최근 정답률과 누적 정답률을 따로 저장합니다.
- 시작 화면에서 선택한 난이도의 `한자 → 읽기`, `한자 → 히라가나 조합`, `읽기 → 한자`, `문장 빈칸 → 한자` 기록을 한 번에 비교할 수 있습니다.
- 예를 들어 N2 읽기→한자 학습을 완료해도 N5·N4 한자→읽기의 회차와 정답률은 바뀌지 않습니다.
- 최근 학습 기록에는 난이도, 학습 모드, 해당 모드 안의 회차, 선택지 수와 정답률을 표시합니다.
- 기록은 계정이나 서버가 아니라 각 사용자의 브라우저 `localStorage`에 저장됩니다.
- 다른 사람에게 기록이 공유되지 않으며, 다른 기기나 다른 브라우저에서는 별도의 기록이 만들어집니다.
- 브라우저의 사이트 데이터를 삭제하거나 시크릿 모드를 종료하면 기록이 사라질 수 있습니다.

## 데이터와 예문 출처

- N5·N4: 능력단어 Word 자료 7~30일차의 단어와 예문 480개
- N3: 제공 PDF에서 정리한 한자 어휘 1,323개
- N3 문장: 제공된 실전모의고사 예문 35개와 [Tatoeba](https://tatoeba.org/ko/) 기반 예문 465개
- N2: 제공 PDF에서 정리한 단어와 예문 100개

Tatoeba 기반 예문은 `CC BY 2.0 FR` 자료이며, 앱 데이터에 원문 링크와 라이선스 정보를 포함합니다. 문제 풀이 후 표시되는 출처 링크에서 원문을 확인할 수 있습니다.

## 사용 환경

최신 Chrome, Edge, Safari, Firefox의 PC 및 모바일 브라우저에서 사용할 수 있습니다. 인터넷 연결은 최초 페이지 접속과 새 버전 갱신에 필요합니다.
