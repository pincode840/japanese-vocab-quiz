# 일본어 단어 맞추기

JLPT N5·N4·N3·N2 단어를 학습할 수 있는 웹 퀴즈입니다.

- 한자 → 읽기
- 읽기 → 한자
- N5·N4·N3·N2 문장 빈칸 → 한자
- 일반 단어와 문장 모드 문제 수 선택
- 선택지 4개·6개·8개 설정
- 키보드 숫자 1~8 선택
- 오답 자동 재출제
- 회차 및 정답률 저장
- 능력단어 Word 예문 360개, N3 문장 빈칸 예문 500개, N2 예문 100개 수록
- N3 PDF 한자 어휘 1,323개 수록
- 문장 모드 단계형 힌트: 첫 오답 후 후리가나, 두 번째 오답 후 문장 해석
