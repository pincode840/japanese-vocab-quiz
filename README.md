# 일본어 단어 맞추기

JLPT N5·N4·N3·N2 어휘를 한자, 읽기, 문장 빈칸 문제로 학습하는 웹 퀴즈입니다. 별도의 설치나 회원가입 없이 브라우저에서 바로 사용할 수 있습니다.

## 바로 실행

[일본어 단어 맞추기 앱 열기](https://pincode840.github.io/japanese-vocab-quiz/)

## 주요 기능

- JLPT `N5·N4`, `N3`, `N2` 난이도 선택
- `한자 → 읽기`, `읽기 → 한자`, `문장 빈칸 → 한자` 모드
- 문제마다 선택지 `4개`, `6개`, `8개` 설정
- 일반 및 문장 문제 수를 `10개부터 해당 데이터 전체`까지 선택
- 키보드 숫자키와 숫자패드 `1~8`로 답 선택
- 정답을 선택하면 단어의 읽기와 뜻 표시
- 오답이면 선택한 답의 단어·읽기·뜻과 올바른 정답 표시
- 틀린 문제는 몇 문제 뒤 자동으로 다시 출제
- 오답 이력이 있는 문제를 다음 회차에서 우선 출제
- 상단 `読` 아이콘을 누르면 언제든지 시작 화면으로 이동

## 문제 수

| 난이도 | 일반 단어 문제 | 문장 빈칸 문제 |
|---|---:|---:|
| N5·N4 | 360개 | 360개 |
| N3 | 한자 어휘 1,323개 | 500개 |
| N2 | 100개 | 100개 |

## 오답 학습 방식

### 일반 단어 모드

틀린 선택지와 함께 해당 선택지의 단어·읽기·뜻을 보여줍니다. 올바른 정답과 뜻도 확인할 수 있으며, 틀린 문제는 잠시 뒤 다시 나옵니다.

예시: `선택한 답 帽子（ぼうし） · 모자`

### 문장 빈칸 모드

1. 첫 번째 오답: 선택한 답의 읽기와 뜻을 표시하고 문장에 후리가나를 붙여 같은 문제에 다시 답하게 합니다.
2. 두 번째 오답: 선택한 답, 정답, 단어 뜻과 문장 해석을 모두 보여줍니다.
3. 두 번 틀린 문장은 몇 문제 뒤 다시 출제됩니다.

## 학습 기록

- 완료 회차, 최근 정답률, 누적 정답률과 최근 학습 기록을 자동 저장합니다.
- 기록은 계정이나 서버가 아니라 각 사용자의 브라우저 `localStorage`에 저장됩니다.
- 다른 사람에게 기록이 공유되지 않으며, 다른 기기나 다른 브라우저에서는 별도의 기록이 만들어집니다.
- 브라우저의 사이트 데이터를 삭제하거나 시크릿 모드를 종료하면 기록이 사라질 수 있습니다.

## 데이터와 예문 출처

- N5·N4: 능력단어 Word 자료의 단어와 예문 360개

## 실행 구조

공개되는 [index.html](./index.html)은 화면, 프로그램 코드와 단어 데이터를 모두 포함한 단일 파일입니다. 사용자의 노트북에 있는 원본 Word나 PDF 파일을 웹에서 불러오지 않으므로 GitHub Pages에서 독립적으로 실행됩니다.
